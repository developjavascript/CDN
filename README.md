# cdn [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> DevJS CDN server

## Installation

```sh
$ npm install --save cdn
```

## Usage

```js
var cdn = require('cdn');

cdn('Rainbow');
```
