orbit
=====
> The DevJS CDN server.

![build status][build-badge]

Orbit is the DevJS CDN server.  It provides JavaScript, CSS, and fonts &em; caching along the way.

## Installation
```shell
$ npm install your-project
```

## Usage
```javascript
import Foo from 'bar';

const foo = new Foo();
console.log(foo.bar);
```
